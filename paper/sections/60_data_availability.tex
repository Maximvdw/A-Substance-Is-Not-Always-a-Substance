\section*{Data Availability}

All data, code, and materials supporting this study are publicly available.
The complete dataset, including the integrated knowledge graph, analysis scripts,
and figures, is archived on Zenodo at \url{https://doi.org/10.5281/zenodo.xxxxxxx}.

The development version of the repository is available on GitHub:
\url{https://github.com/gezever/A-Substance-Is-Not-Always-a-Substance/}.

Machine-readable metadata describing the dataset is provided using the DCAT standard.
All generated data products are published as machine-readable RDF and described using DCAT metadata to support FAIR principles.
